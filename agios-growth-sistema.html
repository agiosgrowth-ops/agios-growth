Configurar Meta Ads
Como obter seu token de acesso:
  1. Acesse developers.facebook.com/tools/explorer
  2. Selecione ou crie um App Meta
  3. Clique em "Generate Access Token"
  4. Adicione a permissão ads_read
  5. Clique em Generate novamente e copie o token
O token padrão dura ~1h. Para 60 dias, gere um Long-Lived Token no Token Debugger.
